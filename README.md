# School_Management
Project for CTU: Create an online portal to track Teachers, Students, Courses, and Grades.

Steps to reproduce...
1. Download and install Docker to your local machine
1. Download and install git to your local machine
1. Download and install npm to your local machine
1. Clone this git repository
1. Run cd .../client
1. Run npm install
1. Run npm audit fix --force
1. cd ..
1. run docker compose up
